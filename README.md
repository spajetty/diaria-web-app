## Overview
Diaria is a web-based diary application designed to help users record, organize, and manage their personal thoughts and daily experiences in a digital format. The system provides essential diary functionalities such as creating, viewing, editing, and deleting diary entries through a simple and user-friendly interface.

Built with modern web technologies, Diaria focuses on accessibility, reliability, and real-time data storage. It serves as a practical platform for personal journaling, academic projects, and digital note-keeping.

### Key Features:
- **Diary Entry Creation**: Users can write and save daily diary entries.
- **Entry Management**: Supports full CRUD (Create, Read, Update, Delete) operations.
- **Real-Time Data Sync**: Diary entries are stored and updated instantly using Firebase.
- **User Authentication**: Secure login and account management using Firebase Authentication.
- **User-Friendly Interface**: Clean and minimal design powered by Bootstrap.
- **Secure Storage**: Cloud-based database ensures data reliability.
- **Responsive Design**: Accessible across different devices and screen sizes.
